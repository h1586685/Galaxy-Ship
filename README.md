Galaxy Ship
===
## ![IMAGE](https://github.com/h1586685/Galaxy-Ship/blob/main/Assets/texture/spaceship.png.png)

Description
---
A 2D side-scrolling shoot'em up game.
+ The game is built with Unity.
+ DEMO : https://www.youtube.com/watch?v=LfhI8pKdE_8

Features
---
+ Edge collider
+ Enemy ships :
  - Player tracking
  - Auto attacking
  - Hit animation
  - Health bar
+ Player ship : 
  - Health bar
  - Bullet bar
+ Round :
  - Round counter
  - Round countdown
  - Break time
